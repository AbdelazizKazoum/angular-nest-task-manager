# Angular + NestJS Task Manager

A full-stack task & project management application built with Angular and NestJS.

## Tech Stack
- Angular
- NestJS
- TypeScript
- JWT Authentication

## Features
- Authentication
- Projects & Tasks management
- Role-based access
- Reactive forms

## Setup

### Frontend
cd frontend
npm install
ng serve

### Backend
cd backend
npm install
npm run start:dev
